# AI 学生评语批量生成器
✨ 一键批量生成个性化、高质量学生期末评语，支持导入导出，老师高效办公工具

## 📌 项目介绍
AI 学生评语批量生成器是一款面向教师、班主任的高效工具，通过 AI 自动根据学生姓名、性格、表现、成绩等信息，生成**自然、真诚、个性化**的期末评语，支持批量处理、一键导出 Excel/CSV，大幅节省老师撰写评语的时间。

## 🎯 核心功能
- ✅ 批量输入学生信息（姓名 + 这次成绩 + 上次成绩 + 出勤表现 + 作业表现），至少提供 **姓名 + 这次成绩**
- ✅ AI 自动生成个性化、温暖、正式的学生评语
- ✅ 支持一键导出 Excel / CSV 文件
- ✅ 纯前端运行，无需后端服务
- ✅ 界面简洁、操作简单、上手即用
- ✅ 适配电脑、平板等多设备使用

学生信息格式示例：
张三  100  全勤
李四，90，85，优

## 👨‍🏫 适用人群
- 中小学班主任
- 各科任课老师
- 教育工作者

## ✨ 项目亮点
- 真正**批量处理**，一次可生成几十~上百条评语
- 评语风格自然、真诚、不生硬
- 零安装、零配置，打开即用
- 完全开源，可自由修改、扩展功能
- 响应式界面，手机/电脑都能用

## 📌 使用场景
- 期末学生评语
- 学期总结
- 学生个性化评价
- 家长通知书文字内容
